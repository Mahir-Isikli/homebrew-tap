# Homebrew tap

```bash
brew install --cask Mahir-Isikli/tap/whoopbar
```

- [**WhoopBar**](https://github.com/Mahir-Isikli/whoopbar) — live WHOOP heart rate in your Mac menu bar, with a local history you own.

These apps are open source and ad-hoc signed (not notarized), so the cask clears the macOS quarantine flag on install and they open without a Gatekeeper warning.
